Motion Detection Game
========

#### Rules ####

The aim of the game is to press space key without being spotted by our computeur.

### Technologie ###

Built with JavaScript using the

```javascript

navigator.mediaDevices.getUserMedia(constraints)

```

function.

## Installing

```cmd
yarn install
```

## Development

* build:

    ```cmd
    gulp
    ```

* serve:

    ```cmd
    gulp serve
    ```
